# 更新日志

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [v1.5.0] - 2026-06-27

### 新增
- Apple 风格设计重构：暖砂岩底色 + 毛玻璃卡片 + 氛围光球
- SF Pro Display 字体，去 emoji 装饰
- spring 动画曲线 `cubic-bezier(0.22, 0.1, 0.2, 1)`
- 页面入场/退场过渡动画

## [v1.3.0] - 2026-06-27

### 变更
- 离线化：SheetJS 内联，消除 CDN 依赖
- 纯浏览器本地运行，零外部网络请求

## [v1.2.0] - 2026-06-26

### 新增
- 工程化加固：错误处理全覆盖
- 20 个测试用例
- PII 注册表扩展
- CDN 预连接优化

### 修复
- 边界情况错误处理完善

## [v1.1.0] - 2026-06

### 新增
- 审核模式：全屏浮水印「内部审核·严禁外传」，30 秒无操作自动退出
- 导出前校验：扫描未脱敏 PII 列，弹出警告
- 7 类 PII 检测：身份证、手机号、姓名、银行卡、邮箱、地址、车牌
- 安全仪表盘：环形覆盖率图 + PII 类型覆盖对比
- 操作日志侧边栏

## [v1.0.0] - 2026-06

### 新增
- 首次发布
- 拖拽上传 Excel/CSV
- 智能 PII 检测（双重检测：列名关键词 + 值抽样验证）
- 三种脱敏模式：部分打码 / 完全打码 / 不脱敏
- 一键导出脱敏文件
- 暗色模式 + 毛玻璃效果
