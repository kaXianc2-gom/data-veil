# 🛡️ DataVeil 数据面纱

> 隐私优先的浏览器端 PII 数据脱敏工具 — 纯本地、单 HTML 文件、零服务器

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Version](https://img.shields.io/badge/version-1.1.0-brightgreen)](https://github.com/kaXianc2-gom/data-veil/releases)

**DataVeil** 是一个纯浏览器端运行的 PII（个人敏感信息）脱敏工具。不需要安装、不需要服务器、不需要注册——下载 HTML 文件，双击即用。所有数据均在本地处理，**绝不上传**。

---

## ✨ 功能全景

### 🔍 智能 PII 检测（7 类 + 自定义）

| PII 类型 | 检测方法 | 技术细节 |
|----------|---------|----------|
| 🆔 身份证号码 | GB 11643 校验码验证 | 加权求和 % 11，支持末位 X |
| 📱 手机号码 | 号段前缀识别 | `1[3-9]` 开头 11 位，自动剥离 +86 |
| 👤 姓名 | 中文模式匹配 | 2-4 字中日韩统一表意文字，排除地名 |
| 💳 银行卡号 | Luhn 算法校验 | 16-19 位，含校验位验证 |
| 📧 电子邮箱 | RFC 5322 正则 | 标准邮箱格式匹配 |
| 🏠 家庭住址 | 关键词组合 | 省份+路街号+市区的多策略匹配 |
| 🚗 车牌号 | 标准格式正则 | 含港澳格式 |

- **双重检测**：列名关键词 + 值抽样验证，置信度 > 50% 自动标记
- **智能识别**：自动推荐最匹配的 PII 类型，支持手动切换

### ⚙️ 灵活脱敏

| 模式 | 效果 | 示例 |
|------|------|------|
| **部分打码** | 保留部分信息，其余替换为 `*` | `138****5678` |
| **完全打码** | 全部替换为 `*` | `***********` |
| **不脱敏** | 保留原始数据 | `13812345678` |

- 每列独立配置检测类型、脱敏模式
- 一键预设：全部打码 / 仅身份证+手机 / 重置

### 🔒 安全机制

- **👁 审核模式**：需主动开启，全屏浮水印「内部审核·严禁外传」，**30 秒无操作自动退出**
- **导出前校验**：扫描未脱敏 PII 列，弹出警告，必须勾选「我已知晓风险」才能导出
- **二次确认**：关闭某列脱敏时弹窗确认
- **操作日志**：侧边栏记录所有关键操作的时间戳

### 📊 数据可视化

- **安全仪表盘**：环形覆盖率图 + PII 类型覆盖对比 + 高风险警告列表 + 一键修复
- **统计报告**：按类型展示发现数量、脱敏数量、覆盖率
- **数据预览**：分页表格，脱敏单元格高亮，审核模式下恢复明文

### 🎨 用户体验

- 🌙 暗色模式 + 毛玻璃效果
- 📂 拖拽上传 .xlsx / .xls / .csv
- 📥 一键导出脱敏文件
- 💾 设置自动记忆
- ⌨️ 键盘快捷键（Esc 退出审核，Ctrl+S 下载）

## 🚀 快速开始

1. **下载** `index.html`（或从 [Release](https://github.com/kaXianc2-gom/data-veil/releases) 下载）
2. **双击** `index.html` — 在浏览器中打开
3. **拖入** Excel / CSV 文件
4. DataVeil **自动检测** PII 列并应用默认脱敏规则
5. **调整**规则（可选）→ 点击 **「下载脱敏文件」**

没有任何安装步骤，不需要命令行，不需要 Python。

## 🧬 生态链

DataVeil 是「数据处理工具链」的第一环：

```
🛡️ DataVeil   →  脱敏清洗
    ↓
📊 [SheetLens](https://github.com/kaXianc2-gom/sheet-lens)  →  智能筛选 + 多维可视化
```

两个工具均遵循**同一设计哲学**：单 HTML 文件、纯本地运行、零服务器依赖。

## 🏗️ 技术架构

| 层级 | 技术 |
|------|------|
| **前端** | 纯 HTML + 原生 JavaScript + CSS |
| **Excel 解析** | [SheetJS](https://sheetjs.com/) 0.18.5 (CDN) |
| **编码检测** | TextDecoder API (UTF-8 / GBK 自动降级) |
| **数据格式** | 单文件，~75 KB，零构建工具 |
| **隐私** | 无后端、无数据库、无遥测，100% 浏览器端 |

## 🔐 隐私声明

- **数据不上传**：所有文件解析、PII 检测、脱敏处理均在浏览器本地内存中完成
- **无网络请求**：除 CDN 加载 SheetJS 库外，不向任何服务器发送数据
- **无持久化存储**：关闭浏览器后所有数据自动清除（仅保留主题偏好等设置）
